Research

  • Pulmonary Artery-Vein Segmentation in 3D Computed Tomography Images

    We propose iSparseUnet, a 3D Unet-like semantic segmentation model that is able to segment tubular structures from pulmonary CT scans of ∼ 33 million pixels in one forward pass within 0.7 second, while achieving a Dice score of ∼ 0.84 on our validation set.

    [More to come]

  • Machine Learning Based FAZ Segmentation in OCTA images [In submission]

    We propose a multi-task learning algorithm for OCT-A image segmentation where the input images are from different domains. The best trained model achieved human-level segmentation performance and only made mistakes on cases where even human experts do not have definite answers.

    [Paper] [Poster] [Patent]

Side Projects

  • Pytorch-minModels

    This is a constantly updated codebase of minimum pytorch implementation for deep learning models. The purpose of this project is to demonstrate the essence of deep learning models by as few code as possible. Please feel free to drop me an email if you'd like me to implement a specific model :)

    [code]

  • Transferring Inductive Bias through Leveled Knowledge Distillation

    We demonstrat that, by distilling knowledge from a convolutional neural networks(CNNs) teacher, a multi-layer perceptron(MLP) can learn a set of weights that mimics CNNs’ translational equivariance. We further propose two approaches to improve the student’s performance and robustness.

    [report] [poster]

  • External Wrench Recovery Using Visual-Tactile Sensors for Robotics Manipulation

    We introduce a dataset for external wrench recoverey from visual-tactile sensor images, and an algorithm to estimate the external forces and torques applied to a robotic manipulator equipped with visual-tactile sensors that combines correspondence matching, classical rendering techniques, and PointNet.

    [report] [slides]

  • Inspecting Ultrasound Image of Unborn Fetus by Deep Learning Integrated System

    We design and train a fined-grained image classifier and a object detector(yolov5) to robustly detect 41 different body parts of unborn fetus from ultrasound images. A website is deployed for doctors and ultrasound operators to directly interact with the models.

    [website(credentials required)]

  • Adaptive Optics-Scanning Laser Ophthalmoscopy Image Analysis Using Deep Learning

    We present both semantic segmentation and object detection techniques to localize and distinguish between two types of photoreceptors from AO-SLO images of human retina. This project later evolved into the RC-UPerNet.

    [report] [slides]

  • Multiplying a sequence of matrices

    Implementation of 2 dynamic programming algorithms for determining the optimal order of multiplying a sequence of matrices introduced by F. Frances Yao's, 1982. They have O(c^n) and O(n^2) time complexity respectively.
